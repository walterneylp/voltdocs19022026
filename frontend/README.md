# Frontend (React + TypeScript)

Estrutura inicial para reconstruir as telas a partir das imagens.

## Estrutura
- `frontend/src/pages`: telas do sistema
- `frontend/src/styles`: estilos específicos por tela
- `frontend/src/components`: componentes reutilizáveis
- `frontend/src/assets`: logos, ícones e imagens
- `frontend/src/components/PreviewNav.tsx`: navegação local de preview
